# hypercube

Interactive 4D Tesseract visualization projected into 3D and then 2D.

## Project Mandates

- **Visual Fidelity:** Maintain smooth, high-frame-rate animations for the hypercube.
- **Interactivity:** Ensure touch and mouse controls remain responsive for all rotation planes (XY, XZ, XW, YZ, YW, ZW).
- **Styling:** Use Vanilla CSS for all UI elements to maintain a lightweight, dependency-free frontend.
- **Deployment:** The project is configured for Firebase Hosting; ensure `public/` remains the distribution directory.
